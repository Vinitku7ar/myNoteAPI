# 🛠️ My First Node.js API

A secure and scalable RESTful API built with **Node.js**, **Express**, **MongoDB**, and **JWT Authentication**.

## 🚀 Features

- ✅ RESTful API structure using Express.js
- 🔐 JWT-based authentication system (Login & Register)
- 🗂️ MongoDB for database (via Mongoose)
- 🔄 CRUD operations for resources
- 🧰 Error handling and validation middleware
- 🌐 CORS and environment-based configuration support
